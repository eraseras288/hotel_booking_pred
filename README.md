# Hotel Bookings Analysis

End-to-end analysis of 119,000+ hotel booking records —
from data cleaning and ML modeling to interactive dashboard.

## Project Structure
- `notebook/` — Python analysis and Logistic Regression model
- `dashboard/` — Power BI interactive dashboard
- `screenshots/` — Dashboard previews

## Notebook Results
| Metric    | Score  |
|-----------|--------|
| Accuracy  | 0.8179 |
| Precision | 0.8118 |
| Recall    | 0.6468 |
| F1-Score  | 0.7200 |
| ROC-AUC   | 0.8945 |

## Dashboard Highlights
- Cancellation rate by month, hotel type, country
- Revenue analysis
- Customer segmentation
- Top booking agents

## Key Insights
- 37% of bookings get canceled
- Non-refundable deposits reduce cancellations significantly
- City Hotel has higher cancellation rate than Resort Hotel
- Longer lead time increases cancellation probability

## Stack
Python, scikit-learn, pandas, matplotlib, Power BI, DAX
